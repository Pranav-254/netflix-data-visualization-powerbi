# Netflix Content Overview: Power BI Data Analysis

This repository features a comprehensive Power BI dashboard providing a strategic overview of Netflix's content library. The project transforms raw data into actionable insights regarding global streaming trends, content distribution, and audience preferences.

## Project Overview
The dashboard is designed to analyze key performance indicators (KPIs) within the Netflix ecosystem, focusing on content growth over time, regional distribution, and genre-based popularity.

## Key Features & Visualizations
* **Content Inventory KPI**: A high-level summary of the total library size, segmented by Movies vs. TV Shows.
* **Global Content Distribution**: A geographic map visualization highlighting which countries are the primary producers of Netflix content.
* **Growth Trends**: Line charts showcasing content additions over the years, identifying peak periods of production.
* **Genre & Rating Analysis**: Breakdown of content by maturity ratings (e.g., TV-MA, PG-13) and popular genres to understand target demographics.
* **Strategic Insights**: Analysis of top directors and cast members contributing to the platform's library.

## Data Source
The analysis is built upon a curated dataset of Netflix titles, including:
* **Metadata**: Title, Director, Cast, and Country.
* **Temporal Data**: Release Year and Date Added.
* **Classification**: Category (Movie/TV Show), Rating, and Genre (Listed In).

## Dashboard Highlights
* **Interactive Slicers**: Filter the entire report by Release Year, Country, or Content Type.
* **Custom Branding**: Incorporates Netflix-themed aesthetics for a professional corporate reporting feel.

## Requirements
* **Power BI Desktop**: To view and interact with the `.pbix` file.
* **Data File**: The report is powered by the underlying `Netflix Overview.pbix` data model.

## Usage
1. Clone this repository.
2. Open `Netflix Overview.pbix` using Power BI Desktop.
3. Use the interactive filters to explore content trends across different regions and years.
